# Endless Worldle (Latest)

This project is a single-file web app.

## Run locally
Open `index.html` in a browser.

> Note: The app loads D3/TopoJSON and map data from CDNs, so you need an internet connection.

## Deploy to GitHub Pages
1. Create a GitHub repo
2. Upload `index.html` to the repo root
3. Repo Settings -> Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
4. Open the published URL on iPhone Safari -> Share -> Add to Home Screen
