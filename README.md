# Endless Worldle (Final w/ Search)

## What you asked for
- Country picker keeps the search bar
- Country **selection behavior matches Mode**:
  - tap once selects
  - tap same item again confirms (or press Confirm)
- Country search is cleared after:
  - Guess
  - Give Up
  - New (new country)
  - Mode change

## Deploy to GitHub Pages
1. Create a GitHub repo
2. Upload `index.html` to repo root
3. Settings -> Pages
   - Source: Deploy from branch
   - Branch: main
   - Folder: / (root)
